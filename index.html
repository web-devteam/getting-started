Getting started

Astro is a virtual assistant website that helps to organize your daily activities. it comprises of feature like

alarms music, text to speech, reminder, notes, timer, calculator, schedule and many more astro comes with three modes, the voice mode, chat mode, and widget or sidebar. the chat mode is the

first screen you see. this will allow you have limited chats with astro and could also be used to give commands. to activate the sidebar type show sidebar. this will allow you have access to most of astros functions

like the alarms and reminders. from here you could set reminders, alarms, add or play music and do a lot more. to activate the voice mode, type turn on voice mode or activate voice mode. this will allow you give commands automatically using voice recognition. you could view alarms, reminders, and play music using the voice mode

there are a lot of ways you could access astro and ways to make them easier this includes adding astro to your desktop as an app. to do so follow the steps below or make use of the video.

secondly adding astro to your browser shortcut using the steps or video

and lastly adding astro to your browsers start up site using the following steps or video

chat mode

the chat mode allows users to have limited chats with astro and can also be used to give commands. Note : astro was not designed to be a chatbot therefore there are many questions he cannot answer or he might give a wrong answer to. in addition astro cannot give response to emoji's so the use of emoji's is not allowed. The chat mode can be used for

  • setting and viewing reminder
  • setting and viewing alarms
  • playing music
  • viewing notes
  • viewing schedule
  • searching the web
  • sending emails
  • opening tabs
  • accessing other modes

to set reminders type "remind me to" reminder "by" reminder time Note : using the chat mode to set reminders only saves the reminder in the first slot. to access order slots use the sidebar. to view a reminder type "read reminder" reminder number e.g "read reminder 1"

to set alarms type "set an alarm for" alarm time. to view an alarm type "view active alarms"

to play music type "play" follow by the music name or title. to play a random song type "play random music". to play your songs type "play music" Note : you can only play songs that have been loaded and saved in the sidebar. use the name that you use to save the song to make reference to the song

to view note type "view" note number. e.g "view note one"

to view your schedule type "read my schedule" and press done to indicate that you have completed your schedule to enable astro stop reminding you

to search the web type "search for" the web search. e.g "search for when did nigeria get it's independence"

to send an email type "send an email to" or "send a mail to" mail address "subject" mail subject "body" mail body

to open a new tab type "open a new tab" and "open youtube" to open youtube

to access other modes type the command to open the mode. for the sidebar type "show sidebar" for the voice mode type "turn on voice mode" and for the timer type "show timer"

voice mode

the voice mode is used to give commands automatically usig your voice. it enables astro to continously listen to the enviroment and sends in what ever sentence that was said without continous tap of a button. if the voice mode should stop listening due to network fluctuation just tap the microphone button to start the process again. to close the voice mode say "close". the voice mode can be used to perform thesame functions as the chat mode. to perform a function say thesame command used to perform that function in the chat mode

sidebar

the sidebar allows users to have more controll over functions. it can be used for tasks like

  • setting alarm
  • setting reminder
  • reading text aloud
  • minor calculations
  • saving notes
  • adding and saving songs
  • editing schedule
  • manipulate settings

to open the music panel type "show sidebar", them tap the music icon. to add songs tap the + button then type the name you would like to save the song as in the name space lastly, add the music file by tapping on choose file and selecting the file. you can't actively play songs from the sidebar, that will require the chat mode.

to open the reminder panel type "show sidebar", them tap the bell icon. to add reminders type the reminder time in the black slot then tap the detail button, this will open the textarea. type the reminder in it and tap the save button. there are 6 different reminder slots that can be reused. note : every reminder setted using the chat mode is saved in the first slot only.

to open the note panel type "show sidebar", them tap the note icon. to add notes tap the detail icon, this will open the textarea, type your note there and press the save button to save. there are 6 note slots that can be reused

to open the text reader type "show sidebar", them tap the speaker icon. type your text in the textarea and tap the read button to read it. this can be used to read stories or copied bible passages and many other texts

to open the schedule panel type "show sidebar", then tap the clipboard icon. to add or change your schedule type your daily schedule in the textarea and press save to save it. your schedule are activities that you do every day.

to open the settings type "show sidebar", them tap the settings icon. the first slot titled username is meant for the user's name while the time slot is meant for the schedule timing. i.e the interval in minutes that would be observered before reminding the user of his schedule note : after saving new settings reload the page to effect changes

to open the alarm panel type "show sidebar", them tap the alarm icon. to add an alarm tap on the add button, this would highlight a slot were you are to put the alarm time. once that is done tap the add button again to add it.
